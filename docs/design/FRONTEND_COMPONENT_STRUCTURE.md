## Color Scheme (Tailwind Classes)

### Primary Colors
```css
--primary-50:  #eff6ff
--primary-100: #dbeafe
--primary-500: #3b82f6
--primary-600: #2563eb
--primary-700: #1d4ed8

/* Usage: bg-primary-600, text-primary-700 */
```

### Semantic Colors
```css
/* Success */
--success: #10b981 (emerald-500)

/* Warning */
--warning: #f59e0b (amber-500)

/* Danger */
--danger: #ef4444 (red-500)

/* Info */
--info: #3b82f6 (blue-500)

/* Urgency Levels */
--critical: #dc2626 (red-600)
--high: #ea580c (orange-600)
--medium: #ca8a04 (yellow-600)
--low: #16a34a (green-600)
```

### Dark Mode
```css
--background: #09090b (zinc-950)
--foreground: #fafafa (zinc-50)
--card: #18181b (zinc-900)
--border: #27272a (zinc-800)
```

/* States: default, hover, active, disabled, focus */

```css

@apply hover:bg-zinc-100 dark:hover:bg-zinc-800
```

```css
/* Accessibility */
@apply focus:ring-2 focus:ring-primary-500 focus:outline-none
```

## Animation Classes

```css
/* Fade In */
@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}
.animate-fade-in

/* Slide In */
@keyframes slideIn {
  from { transform: translateX(-100%); }
  to { transform: translateX(0); }
}
.animate-slide-in

/* Pulse (for loading) */
@keyframes pulse {
  0%, 100% { opacity: 1; }
  50% { opacity: 0.5; }
}
.animate-pulse
```
